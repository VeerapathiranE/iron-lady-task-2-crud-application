## Task 2: Internal Business Automation (CRUD Application)

### Tool Used
- Glide

### Description
An internal Lead Management System was developed to support Iron Lady’s internal business workflow.
The application helps manage inquiries and potential learners efficiently.

### Features
- Create lead records
- View lead details
- Update lead information
- Delete lead records

### Business Value
- Replaces manual tracking (Excel / notes)
- Improves operational efficiency
- Centralized lead management
This project was built using AI-powered and no-code platforms as encouraged in the assignment.
Instead of traditional source code, this repository contains documentation, workflow explanations, screenshots, and demo videos demonstrating the final working output.
